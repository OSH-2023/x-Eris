# x-Eris
The course project of the Operating Systems (H), USTC, 2023.

## 📄 项目简介 Description
我们希望在基于现有的FreeRTOS的虚拟文件系统模块FreeRTOS-Plus-Fat进行兼容性拓展及安全性能优化，得到一个支持全面，安全快捷的的嵌入式系统VFS。

项目名称为：Eris。
